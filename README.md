## Hello, I'm Emmanuel Yokoyama!

<img align="right" src="https://media.tenor.com/PLIr_VkF6ywAAAAM/ghostedvpn-hacker-cat.gif" alt="" width="200"/>

### 4th Semester Software Development Multiplatform Student at *Fatec São José dos Campos - Prof. Jessen Vidal*  

I have always been interested in technology, initially influenced by my family, as I have relatives working in the field. During my childhood, I had my first contact with programming by developing modpacks for the game Minecraft, working on basic features with low complexity.  

In high school, I enrolled in a technical program in Informatics alongside my regular education, which provided me with a strong foundation and was where I decided to pursue this career. After graduating, I discovered Fatec and its teaching approach, and saw it as a great opportunity to evolve, improve my technical skills, and grow professionally.  

My main gains in both hard and soft skills came through APIs (*Integrated Project-Based Learning*), developed in teams. This experience strengthened my collaboration, problem-solving, and critical thinking skills. During this period, I had the opportunity to learn, apply, and improve my knowledge in several technologies, such as Python with Flask, JavaScript, TypeScript, Docker, Cloud services, Java with Spring Boot, React, MySQL, MongoDB, and project management using Jira, both in academic and personal projects.  

Additionally, I have experience with **CI/CD and DevOps practices**, working with **AWS services such as EC2 (running on RHEL), RDS with PostgreSQL**, and automation pipelines using tools like GitHub Actions, focusing on deployment, scalability, and infrastructure management.  

Beyond technical skills, I also developed interpersonal skills and experience with agile methodologies, acting both as a developer and as a Product Owner within development teams.
